# Collaboration avec des écoles

## Description
Fournissez des pièces éducatives pour des projets scolaires ou universitaires.

## Exemples
- Pièces pour des projets de robotique.
- Modèles pour des cours de sciences (ex : molécules, organes).
- Accessoires pour des projets artistiques.

## Site d'exemple
[Tinkercad for Education](https://www.tinkercad.com/education)

## Astuces
- Contactez les enseignants ou les responsables de projets éducatifs.
- Proposez des tarifs préférentiels pour les établissements scolaires.
- Offrez des conseils pour intégrer l'impression 3D dans les programmes éducatifs.
