# Services d'impression 3D

## Description
Proposez vos services pour imprimer des fichiers 3D envoyés par des clients.

## Exemples
- Impression de modèles envoyés par des designers.
- Impression de pièces pour des projets DIY.
- Impression de cadeaux personnalisés.

## Site d'exemple
[3D Hubs](https://www.hubs.com/)

## Astuces
- Créez un site web ou une page sur les réseaux sociaux pour promouvoir vos services.
- Offrez des tarifs dégressifs pour les grosses commandes.
- Proposez des conseils pour optimiser les modèles avant impression.
