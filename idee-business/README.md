# Idées Business

Ce dépôt contient des idées et concepts pour des projets entrepreneuriaux.

## Structure

- `idees/` - Dossier contenant des fichiers Markdown avec des idées détaillées
- `recherche/` - Recherches de marché et analyses
- `business-plans/` - Modèles et plans d'affaires

## Comment contribuer

1. Créez un nouveau fichier dans le dossier `idees/` avec un nom descriptif
2. Utilisez le modèle ci-dessous pour structurer votre idée
3. Faites une pull request pour ajouter votre idée

## Modèle d'idée

```markdown
# Titre de l'idée

## Description
Description détaillée de l'idée

## Marché cible
À qui cette idée s'adresse-t-elle ?

## Avantages concurrentiels
Ce qui rend cette idée unique

## Modèle économique
Comment cette idée générera des revenus

## Prochaines étapes
Actions à entreprendre pour valider cette idée
```
